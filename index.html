depjs

JavaScript dependency manager.

Download as .zip Download as .tar.gz View on GitHub

depjs is a JavaScript dependency manager.

Syntax

Configure your project and if you use any non-depjs lib, you can add rules as they are attached to depjs for further use.

dep.config({
  project: 'depjs',
  version: '0.1.0',
  path: '/js/',
  vendor: {
    underscore: '_',
    jquery: 'jQuery',
    backbone: {
      use: ['vendor.underscore', 'vendor.jquery'],
      attach: function (_, $) {
        return Backbone;
      }
    }
  }
});

Define your own depjs module, which may depend on other libs.

dep.use(['vendor.jquery'], function ($) {
  alert(123);
}, 'lib.graph');
